## Getting Started

### Creating Your First Mockup

1. **Open FrameIt**: Visit [frameit.social](https://frameit.social) or open the app locally
2. **Upload an Image**: 
   - Click "Start Creating" button
   - Or drag and drop an image onto the canvas
   - Or click the upload icon in the gallery
3. **Customize**: Use the control panels on the right to adjust your mockup
4. **Export**: Click the export button when you're satisfied with your design

### Understanding the Interface

The FrameIt interface consists of three main areas:

- **Left Sidebar (Gallery)**: Manage multiple canvas designs
- **Center Canvas**: Preview your mockup in real-time
- **Right Sidebar (Controls)**: Customize every aspect of your mockup

## Image Management

### Uploading Images

**Supported Formats**: PNG, JPEG, WebP, and other major image formats

**Size Limits**:
- Maximum file size: 10MB per image
- Minimum dimensions: 50×50 pixels
- Maximum dimensions: 8000×8000 pixels

**Upload Methods**:
1. Click "Start Creating" or upload button
2. Drag and drop images onto the canvas
3. Click the "+" button in the gallery
4. Click on image slots in multi-image layouts

### Managing Multiple Images

- **Add Images**: Click empty slots in your chosen layout
- **Replace Images**: Click on filled slots to replace
- **Remove Images**: Click the trash icon on image slots
- **Select Images**: Click slots to select which image to edit
- **Reorder Images**: Use the move/duplicate options in slot menus

### Image Transformations

**Flip Controls**:
- Horizontal Flip: Mirror image left-to-right
- Vertical Flip: Mirror image top-to-bottom

**Rotation**:
- Range: -180° to +180°
- Use slider or enter exact value
- Reset button returns to 0°

**Pan Controls** (when Smart Fill is enabled):
- Pan X: Move image horizontally (-100 to +100)
- Pan Y: Move image vertically (-100 to +100)
- Useful for repositioning images within the frame

**Smart Fill**:
- Toggle to fill the entire canvas with your image
- Maintains aspect ratio while filling space
- Combine with pan controls for precise positioning

## Layout Selection

FrameIt offers 8 professional layout templates:

### Single Image
- **Best for**: Individual screenshots, hero images
- **Images**: 1
- **Use case**: App store screenshots, portfolio pieces

### Side by Side
- **Best for**: Before/after comparisons, feature showcases
- **Images**: 2 (equal width)
- **Use case**: Product comparisons, dual views

### Top & Bottom
- **Best for**: Sequential steps, vertical comparisons
- **Images**: 2 (equal height)
- **Use case**: Process flows, mobile app screens

### Main + Sidebar
- **Best for**: Feature highlights with context
- **Images**: 2 (one large, one small)
- **Use case**: Main feature with detail view

### Three Columns
- **Best for**: Feature trios, step-by-step guides
- **Images**: 3 (equal width)
- **Use case**: Three-step processes, feature sets

### 2×2 Grid
- **Best for**: Feature grids, app screen collections
- **Images**: 4 (equal size)
- **Use case**: App screen showcases, feature grids

### Hero + Thumbnails
- **Best for**: Main feature with supporting images
- **Images**: 4 (one large, three small)
- **Use case**: Product showcases, app presentations

### L-Shape
- **Best for**: Asymmetric designs, creative layouts
- **Images**: 3 (one large, two small)
- **Use case**: Creative portfolios, unique presentations

- Click any layout thumbnail to switch
- Images are preserved when switching (up to layout limit)
- Extra images are removed if new layout has fewer slots

### Background Effects

Transform your backgrounds with powerful effects:

#### Blur Effect
- **Range**: 0-50px
- **Use**: Soften backgrounds, create depth of field
- **Tip**: 10-20px works well for most designs

#### Color Adjustments
- **Saturation** (0-200%): Increase or decrease color intensity
  - 0%: Grayscale
  - 100%: Original
  - 200%: Hyper-saturated
- **Hue Rotation** (0-360°): Shift colors around the color wheel
  - Great for color variations
  - 180° creates complementary colors
- **Contrast** (0-200%): Adjust difference between light and dark
  - <100%: Softer, muted
  - >100%: Bolder, more dramatic
- **Brightness** (0-200%): Lighten or darken
  - <100%: Darker
  - >100%: Lighter

#### Distortion Effects
- **Twirl**: Spiral distortion from center
  - Creates dynamic, swirling patterns
  - Best with gradients
- **Wave**: Horizontal wave distortion
  - Adds movement and flow
  - Subtle effect works best
- **Ripple**: Circular ripple from center
  - Water-like effect
  - Great for organic feels
- **Zoom**: Scale background from center
  - 100%: Original size
  - <100%: Zoomed out
  - >100%: Zoomed in
- **Shake**: Random vibration effect
  - Adds energy and motion
  - Use sparingly
- **Lens**: Barrel/pincushion distortion
  - Fish-eye lens effect
  - Creates unique perspectives

**Effect Tips**:
- Combine multiple effects for unique looks
- Keep effects subtle for professional designs
- Bold effects work well for creative projects
- Reset individual effects with reset buttons

## Texture & Noise Overlays

Add depth and character with texture overlays.

### Available Textures
- **Paper**: Classic paper texture
- **Canvas**: Fabric canvas texture
- **Grain**: Film grain effect
- **Fabric**: Woven fabric texture
- And more...

### Texture Controls

**Intensity** (0-100%):
- Controls the strength of the texture pattern
- 100% = Full texture visibility
- 50% = Subtle texture
- 25% = Very subtle

**Opacity** (0-100%):
- Controls overall transparency of the overlay
- 100% = Fully opaque
- 50% = Semi-transparent
- 25% = Very subtle

**Scale** (50-200%):
- Controls the size of the texture pattern
- 100% = Original size
- 50% = Smaller, tighter pattern
- 200% = Larger, more spread out

**Blend Mode**:
- **Multiply**: Darkens, great for adding depth
- **Overlay**: Balanced blend, most versatile
- **Screen**: Lightens, creates glow
- **Soft Light**: Subtle, gentle blend
- **Hard Light**: Strong, dramatic blend
- **Color Dodge**: Brightens highlights
- **Color Burn**: Darkens shadows

**Invert**:
- Flips the texture pattern
- Light becomes dark, dark becomes light
- Creates different texture effects

### Random Noise
- Click the dice icon for instant random texture
- Randomly selects texture type and settings
- Great for quick styling

### Texture Tips
- Start with 30-50% opacity for subtle effects
- Use multiply blend mode for vintage looks
- Use overlay for balanced texture
- Match texture scale to your design size
- Combine with background effects for unique looks

## Frame & Shadow Controls

Perfect your mockup's presentation with frame and shadow settings.

### Corner Radius
- **Range**: 0-100px
- **0px**: Sharp corners (screenshots, technical docs)
- **10-25px**: Subtle rounding (modern, professional)
- **50-100px**: Heavy rounding (friendly, approachable)
- **Tip**: Match your brand's design language

### Padding
- **Range**: 0-200px
- **0px**: No padding, edge-to-edge
- **50-100px**: Comfortable spacing (recommended)
- **150-200px**: Generous spacing, focus on image
- **Tip**: More padding = more background visible

### Shadow Customization

#### Shadow Opacity
- **Range**: 0-100%
- **0%**: No shadow
- **15-30%**: Subtle, professional (recommended)
- **50-100%**: Strong, dramatic

#### Shadow Blur
- **Range**: 0-50px
- **0px**: Hard shadow edge
- **5-15px**: Soft, natural shadow (recommended)
- **30-50px**: Very soft, diffused

#### Shadow Offset
- **X Offset** (-50 to +50px): Horizontal position
  - Negative: Shadow to the left
  - Positive: Shadow to the right
  - 0: Centered
- **Y Offset** (-50 to +50px): Vertical position
  - Negative: Shadow above
  - Positive: Shadow below (recommended)
  - 0: Centered

#### Shadow Color
- Default: Black (#000000)
- Use color picker for custom colors
- Colored shadows create unique effects
- Match shadow color to background for cohesion

#### Shadow Position Grid
- Click any of the 9 positions for instant shadow placement
- Center: No offset (glow effect)
- Corners: Diagonal shadows
- Sides: Directional shadows
- **Tip**: Bottom-right is most natural (light from top-left)

### Quick Reset Buttons
- Each control has a reset button
- Returns to default value
- Saves time when experimenting

### Frame & Shadow Tips
- **Professional Look**: 25px radius, 75px padding, 15% opacity, 8px blur, 0/4px offset
- **Dramatic Look**: 0px radius, 50px padding, 30% opacity, 20px blur, 0/10px offset
- **Minimal Look**: 15px radius, 100px padding, 10% opacity, 5px blur, 0/2px offset
- **Floating Effect**: High blur (30px+), low opacity (10-15%), centered or slight offset

- **Contrast**: Ensure text is readable against background
- **Hierarchy**: Use size and weight to create visual hierarchy
- **Spacing**: Don't overcrowd - less is more
- **Consistency**: Use 2-3 fonts maximum

## Export Options

Export your mockups in various formats and sizes.

### Export Formats

#### PNG (Recommended)
- **Best for**: Screenshots, graphics with transparency
- **Quality**: Lossless, perfect quality
- **File Size**: Larger than JPEG
- **Use when**: Quality is priority, need transparency

#### JPEG
- **Best for**: Photos, web images
- **Quality**: Adjustable compression
- **File Size**: Smaller than PNG
- **Use when**: File size matters, no transparency needed

#### WebP
- **Best for**: Modern web use
- **Quality**: Better compression than JPEG
- **File Size**: Smallest
- **Use when**: Modern browsers only, web optimization
- **Note**: Not supported in all browsers

### Resolution Presets

#### Social Media Optimized

**1080×1080 (Square)**
- **Platforms**: Instagram, Facebook, LinkedIn, Pinterest
- **Best for**: Feed posts, profile images
- **Aspect Ratio**: 1:1

**1080×1350 (Portrait)**
- **Platforms**: Instagram, Facebook, LinkedIn, Pinterest
- **Best for**: Instagram feed posts
- **Aspect Ratio**: 4:5

**1080×1920 (Story)**
- **Platforms**: Instagram, Facebook, Snapchat, TikTok
- **Best for**: Stories, reels, vertical video thumbnails
- **Aspect Ratio**: 9:16

**1200×627 (LinkedIn Post)**
- **Platforms**: LinkedIn
- **Best for**: LinkedIn article images, link previews
- **Aspect Ratio**: 1.91:1

**1200×800 (Substack)**
- **Platforms**: Substack, blogs
- **Best for**: Newsletter headers, blog posts
- **Aspect Ratio**: 3:2

**1920×1080 (Landscape HD)**
- **Platforms**: Facebook, LinkedIn, YouTube
- **Best for**: Cover images, video thumbnails
- **Aspect Ratio**: 16:9

### Size Options

#### Original Size (1x)
- Export at canvas resolution
- Standard quality
- Balanced file size

#### Double Size (2x)
- Export at 2x canvas resolution
- High-DPI displays (Retina)
- Larger file size
- **Use for**: Print, high-quality displays

#### Half Size (0.5x)
- Export at 0.5x canvas resolution
- Web optimization
- Smaller file size
- **Use for**: Web thumbnails, email

#### Custom Size
- Enter exact width and height
- Aspect ratio lock option
- **Use for**: Specific requirements

### Quality Settings

**High Quality**
- Best quality, larger file size
- Recommended for final exports
- JPEG quality: 100%

**Medium Quality**
- Balanced quality and file size
- Good for most uses
- JPEG quality: 80%

**Low Quality**
- Smaller file size, lower quality
- Web optimization
- JPEG quality: 50%

### Export Workflow

#### Single Export
1. Click "Export" button
2. Choose format (PNG/JPEG/WebP)
3. Select size option
4. Choose quality (for JPEG/WebP)
5. Click "Export"
6. File downloads automatically

#### Batch Export (Export All)
1. Click "Export All" button
2. Choose format and settings
3. All canvases export with sequential numbering
4. Files download as: frameit-mockup-1.png, frameit-mockup-2.png, etc.

### Export Tips
- **PNG for Quality**: Use PNG for final, high-quality exports
- **JPEG for Web**: Use JPEG for web use, smaller files
- **2x for Retina**: Export at 2x for high-DPI displays
- **Batch Export**: Save time by exporting all at once
- **File Naming**: Files are automatically named with timestamp
- **Preview First**: Always preview before exporting
- **Test Sizes**: Export at different sizes to find optimal balance

## Troubleshooting

### Common Issues

#### Image Won't Upload
**Problem**: Image fails to upload
**Solutions**:
- Check file size (max 10MB)
- Verify file format (PNG, JPEG, WebP)
- Check image dimensions (50-8000px)
- Try a different browser
- Clear browser cache

#### Export Not Working
**Problem**: Export button doesn't work
**Solutions**:
- Check browser compatibility
- Allow downloads in browser settings
- Try different export format
- Disable browser extensions
- Try incognito/private mode

#### Canvas Appears Blank
**Problem**: Canvas shows nothing
**Solutions**:
- Upload an image first
- Check if image is hidden
- Refresh the page
- Clear browser cache
- Check browser console for errors

#### Text Not Visible
**Problem**: Text layer doesn't appear
**Solutions**:
- Check text color vs background
- Verify opacity is not 0%
- Check if layer is hidden
- Verify z-index setting
- Check text size

#### Slow Performance
**Problem**: App is slow or laggy
**Solutions**:
- Reduce image sizes before upload
- Close other browser tabs
- Clear browser cache
- Use fewer effects simultaneously
- Restart browser

### Browser Compatibility

#### Recommended Browsers
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

#### Known Limitations
- WebP export not available in older browsers
- Some effects may perform slower on mobile
- Canvas size limited by device memory

### Getting Help

If you encounter issues not covered here:

1. **Check Browser Console**: Press F12 to see error messages
2. **Try Different Browser**: Test in another browser
3. **Clear Cache**: Clear browser cache and cookies
4. **Contact Support**: Email hello@frameit.social
5. **Report Bug**: Open issue on GitHub
