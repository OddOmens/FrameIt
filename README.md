# FrameIt

**FrameIt** is an open-source tool designed to help you transform your screenshots and images into professional, stunning mockups. Whether you're a developer, designer, or creator, FrameIt provides a suite of styling tools to make your content shine.

## Features

-   **Custom Layouts**: Organize multiple images in grids, rows, or columns.
-   **Professional Styling**: Add realistic shadows, rounded corners, and padding.
-   **Backgrounds**: Choose from curated gradients, solid colors, or create your own.
-   **Effects**: Apply noise, grain, and other textures to add depth.
-   **Typography**: Overlay text with customizable fonts and styles.
-   **Export**: Download your creations in high-resolution formats suitable for social media and portfolios.
